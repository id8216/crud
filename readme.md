CRUD приложение

Стандартное crud приложение, которое отображает 
список всех юзеров в базе (с пейджингом).
С возможностью их удаления, редактирования,
добавления новых и поиском по уже существующим.

Автор: Иван Диброва

При разработке приложения использовались:
   - Intellij IDEA 2016.2.2 Ultimate Edition
   - MySQL WorkBench 6.3.6 Community
   - Maven 3.3.9 
   - Apache Tomcat 8.5.3

База данных создавалась в MySQL WorkBench из файла create_database.sql

Запуск приложения проводился через Apache Tomcat из Intellij IDEA.
